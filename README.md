# EECS16b Course Reader

https://eecs16b.streamlit.app/


Having trouble navigating through the extense EECS16B resources provided by staff? Use https://eecs16b.streamlit.app/ to "command-f" through all content, powered by semantic search.

Tech Stack:
- Open AI's Text Embeddings API
- Streamlit for hosting
- CSV for database


Looking to add support for a vector store in the future (ChromaDB, Pinecone, etc).

Looking to add support for in-line LaTeX rendering of relevant course content.
